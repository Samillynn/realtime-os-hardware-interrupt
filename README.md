# Build Instructions
## download
```
git clone https://git.uwaterloo.ca/h44ding/cs452_k3.git
```
## build
```bash
cd cs452_k3
make clean && make
```
## run
put Pi number as the parameter for the ./upload.sh

for example, CS017542 (Pi No.2):
```bash
./upload.sh 2
```
